# Hands Together for Opera — Concert Tonight

Phone-first digital program for the Millar Piano Duo concert on July 18, 2026, at Remenyi House of Music (Toronto).
Deployed site: https://raposamillar.github.io/concert-tonight

## Pages

- `index.html` — landing / menu
- `program.html` — program order
- `bios.html` — performer and composer bios
- `notes.html` — program notes
- `sponsors.html` — presenter and sponsors

## Stack

Static HTML and CSS only (no build step, JavaScript only if needed later).

## Notes

- Designed for phone use before and after the concert
- Shared styles live in `styles.css`
- HTML/CSS structure was created with assistance from Cursor; concert content is by the artists and presenters.
